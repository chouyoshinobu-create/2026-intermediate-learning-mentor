# Codex Project Instructions

這是「2026中級學習導師專案」，主要用來整理中級日語 L1-L15 的教材、測驗、簡報、PDF、Excel 題庫與繳交資料。

## 工作原則

- 專案本體放在本機 `~/Projects/2026中級學習導師專案`，不要放 OneDrive。
- 使用 GitHub 同步 Air、Pro、Mac mini M4、Mac mini M2 等電腦。
- 每次開始工作前先 pull，每次換電腦前 commit + push。
- 不要同時在兩台電腦編輯同一份 Word、Excel、PowerPoint 或 PDF。
- 保留原始教材與輸出成品的區分，不要任意覆蓋原檔。
- 大型原始檔若超過 GitHub 限制，放本機或雲端硬碟，不直接提交到 Git。

## 專案記憶

- `AGENTS.md`：給 Codex 看的長期規則。
- `PROJECT_LOG.md`：記錄已做決定、重要操作、跨電腦同步狀態。
- `TODO.md`：記錄下一步要做的工作。

Codex 在任何電腦打開此專案時，請先閱讀這三個檔案，再繼續工作。
